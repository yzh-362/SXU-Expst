# Typst Experiment Report Template

这是一个基于 [Typst](https://typst.app/) 编写的现代化、优雅的高校实验报告模板。相比于传统且笨重的 LaTeX 模板，它具有编译速度极快（几乎实时预览）、语法更接近 Markdown 且清晰易读、极易定制等显著优势。无论您是撰写物理、化学实验，还是计算机课程设计，本模板都能让您将精力完全集中在报告内容本身，而无需为了版面格式和宏包冲突焦头烂额。

## 特性
* **Skills文档支持**
  现在已经为模板一开头添加了一个 Skills，您可以将这个文件交给AI，这个文件的开头有一段skills描述，写了这个模板文件各个函数的使用方法，通过这个Skills，它会帮您快速生成符合规范的实验报告，大大提升了使用便捷性。
* **自动化且严谨的表头生成**：采用底层 `grid` 与 `box` 结合的强力布局引擎。只需在文档开头或配置文件中填入参数，即可自动生成格式规范、下划线严格对齐的实验报告表头，彻底避免了每次手调空格对齐的繁琐，确保多份报告的风格高度统一。

* **个人信息自动注入 (新增)**：支持通过统一的 `config.toml` 文件集中管理姓名、学号、专业等个人信息。一次配置，随处可用，免去每次新建报告时重复输入的麻烦。

* **智能日期处理与回落机制**：支持自动获取编译当天的当前系统日期。这在赶交报告时非常实用，避免了直接复制旧报告导致忘记修改提交日期的尴尬情况。当然，您也可以通过参数手动固定特定日期。

* **Logo 灵活适配与占位**：左上角的 Logo 支持自定义路径，可自由调整大小，且自动在保留的左侧间隙中绝对水平居中。如果暂时没有准备好学校的 Logo 图片，模板也会优雅地生成一个带有描边的圆形 `[Logo]` 占位符，不会导致页面布局崩塌。

* **开箱即用的中英双语与本地化**：默认配置了支持中英文混排的字体回落机制（优先使用 Times New Roman 渲染西文，配合 SimSun/宋体 渲染中文），免去了 LaTeX 中配置繁琐的中文字体映射过程。同时，自动将内置的 "Figure" 和 "Table" 本地化翻译为 "图" 和 "表"，符合国内高校的学术排版规范。

* **现代化的代码块优化**：内置了优雅的代码高亮展示样式，自动为代码块添加浅灰色背景、平滑圆角和精细边框，配合特定的等宽字体，大幅提升代码阅读体验。这对于需要附录 Python、C++ 或 MATLAB 脚本的理工科实验报告尤为贴心。

* **优雅的双栏排版支持**：提供了一种灵活的局部双栏布局代码片段示范。Typst 的局部双栏允许您在页面的任意位置开启，非常适合用于图文并排解释、或是密集的实验数据对比展示，区块结束后又会自动恢复单栏，提供极高的版面自由度。

* **全能原生数据可视化组件 (进阶)**：深度集成了基于 CeTZ 的绘图函数。您可以通过极其简单的接口，在同一图表中实现 **直方图、数据散点、原始折线、Catmull-Rom 平滑插值曲线、线性回归拟合线、甚至纯数学连续理论曲线** 的全方位无损叠加渲染。此外还内置了矩阵热力图组件。

* **表头页面级控制**：采用 Typst 官方推荐的单页普通流布局机制。通过在文档主入口精确控制外边距（`margin`）并单独作为一个 `block` 容器，它不仅能完美作为第一页的封面表头，还能通过精准设定的底层间距自动推开下方的正文，实现极具美感的留白。

## 快速开始

### 1. 本地环境或在线环境准备

您可以使用以下两种方式之一来编译本模板：
* **Typst 官方在线编辑器**：访问 [Typst Web App](https://typst.app/)，注册账号后新建一个空白项目（Blank Project）。将所需的 `.typ` 文件、`config.toml` 和图片直接拖拽上传到网页左侧的文件树中即可实时预览。
* **VsCode 插件**：在 Visual Studio Code 中安装Tinymist插件。打开本项目文件夹后，直接点击 `main.typ` 文件右上角的 "预览" 按钮，即可在编辑器内侧边栏实时预览文档。
* **在线 Web App (推荐新手)**：直接访问 [Typst Web App](https://typst.app/)，注册账号后新建一个空白项目（Blank Project）。将所需的 `.typ` 文件、`config.toml` 和图片直接拖拽上传到网页左侧的文件树中即可实时预览。

* **本地命令行工具**：在终端通过包管理器安装 Typst CLI（例如 `winget install typst` 或 `brew install typst`）。然后在终端运行 `typst watch main.typ`，它会在您每次保存时瞬间增量编译出 PDF。

### 2. 个人信息配置 (推荐)

在项目根目录下创建一个名为 `config.toml` 的文件，填入您的固定信息：

```toml
# config.toml
name = "您的姓名"
student-id = "您的学号"
college = "您的学院"
major = "您的专业"
```

### 3. 引入模板与正文编写

将本项目中的 `report-template.typ` 文件以及您的学校 Logo 图片（例如 `logo.png`）放在同一个目录下。
新建一个正文文件（例如 `main.typ`），引入模板并读取配置：

```typst
#import "report-template.typ": *

// 读取同目录下的配置文件
#let my-info = toml("config.toml")

#show: experiment-report.with(
  college: my-info.college,
  major: my-info.major,
  student-id: my-info.student-id,
  name: my-info.name,
  course: "操作系统原理实验",
  location: "九教 402 机房",
  // year: "2026",  // 若注释掉这三行日期参数，则自动获取系统当前时间
  // month: "3",    
  // day: "19",     
  project: "进程调度算法的模拟与实现",
  // logo: "logo.png" // 若有校徽可在此处传入路径
)

= 实验目的
在这里开始编写您的实验内容。使用单个等号 `=` 表示一级标题，两个等号 `==` 表示二级标题...
```

## 核心绘图组件库说明

本模板内置了极其强大的原生数据分析与绘图宏，无需引入额外依赖即可直接在正文中调用：

### 数据计算

```typst
// 简单线性回归，返回包含 k(斜率), b(截距), r2(决定系数) 的字典
#let reg = linear-regression(x-list, y-list) 
```

### 综合折线与曲线绘图 (`simple-line-plot`)

这是一个万能绘图函数，支持多图层叠加。

```typst
#simple-line-plot(
  x-list, y-list,                  // 传入数据，如果不传则进入纯数学曲线模式
  width: 10cm, height: 6.5cm,      // 自定义宽高
  smooth: true,                    // 开启 Catmull-Rom 平滑插值
  show-raw-line: true,             // 叠加原始数据点连接的虚线
  regression-data: reg,            // 叠加线性回归拟合线 (传入上方的 reg 变量)
  curve-func: x => calc.sin(x),    // 叠加自定义的数学连续函数曲线
  bar-data: bar-list,              // 在最底层自动置底渲染直方图
  title: "图表标题",
  x-label: [浓度 ($"mol"\/"L"$)],   // 坐标轴标签，注意物理化学单位的斜杠必须用 \/ 转义
  y-label: [吸光度 ($"A"$)]
)
```

### 辅助图表

```typst
// 矩阵热图
#simple-heatmap(matrix-data)

// 独立直方图
#simple-bar-chart(data-list)

// 蓝底提示框 (用于排版思考题等)
#question-box(title: "实验思考题")[内容区域]
```

## 参数说明

`experiment-report` 函数接受以下参数进行初始化：

| 参数名 | 类型 | 说明 | 默认值 | 
| ----- | ----- | ----- | ----- | 
| `college` | String | 学院名称，如 "理学院" | `""` | 
| `major` | String | 专业名称，如 "应用物理学" | `""` | 
| `course` | String | 实验课程名称 | `""` | 
| `location` | String | 实验地点或实验室编号 | `""` | 
| `year` | String / Integer | 实验年份。传入 `none` 则自动获取当前年份 | `none` | 
| `month` | String / Integer | 实验月份。传入 `none` 则自动获取当前月份 | `none` | 
| `day` | String / Integer | 实验日期。传入 `none` 则自动获取当前日期 | `none` | 
| `student-id` | String | 学号 | `""` | 
| `name` | String | 您的姓名 | `""` | 
| `project` | String | 本次具体实验的项目名称 | `""` | 
| `logo` | String / Image | 放置于左上角的 Logo。可传入文件路径字符串，或直接传入 `image` 对象。 | `"/typst-img/1773924825701.png"` | 

## 自定义修改建议

如果您需要根据所在学校的具体格式要求（如特定的字号、特定的标题名称）进行微调，可以直接修改模板源码 `report-template.typ` 中的以下核心部分：

* **更改主标题名称**：如果您不需要叫做“实验报告”，可以在源码的 `grid` 布局中找到 `text(...)[实验报告]` 这一行，将其替换为 `[课程设计报告]` 或 `[上机实习报告]`。

* **Logo 大小调整**：在 `experiment-report` 函数定义的 `grid` 布局内，找到 `let logo-size = 2.5cm`，适度增大或缩小该数值即可等比例调整 Logo 宽度，它会自动保持居中。

* **页边距与页面容器间距**：如果学校要求更严格的页边距，可以在 `set page` 函数内，通过修改 `margin: (top: 1cm, bottom: 1cm, left: 1cm, right: 1cm)` 调整整体外边距。同时可以调整包裹表头的 `block(height: 4.5cm, ...)` 参数，以控制表头区域的高度。

* **全局中英文字体设置**：修改源码中的 `#set text(font: ("Times New Roman", "SimSun"), size: 12pt)`。您可以根据操作系统的不同，将其替换为您偏好的学术字体组合。
